# Установка Voice Vault Search

Плагин для Obsidian: говоришь вопрос вслух → он распознаётся → открывается нужная заметка.

---

## Что понадобится

| Что | Зачем | Обязательно? |
|-----|-------|--------------|
| **Obsidian** (Windows) | сам плагин | да |
| **Python 3.10+** | локальный движок распознавания | да |
| **Ollama** | вытаскивает вопрос из речи + генерит «Возможные вопросы» (сильно точнее ищет) | очень желательно |
| **NVIDIA GPU** | ускорение | нет (без него работает, но медленнее) |

**Нужна ли Ollama?** Строго говоря — нет: распознавание и поиск заведутся и без неё.
**Но с ней ощутимо лучше** — она (1) выдёргивает сам вопрос из разговора, отбрасывая
болтовню, и (2) генерит для заметок блоки «Возможные вопросы», по которым поиск попадает
точнее. Рекомендую поставить.

---

## Установка по шагам

### 1. Python
Поставь Python 3.10+ с [python.org](https://www.python.org/downloads/).
На Windows **обязательно** галочка «**Add Python to PATH**» при установке.

### 2. Плагин в Obsidian
Скопируй папку `voice-vault-search` в `<твой-vault>/.obsidian/plugins/voice-vault-search/`.
В Obsidian: **Settings → Community plugins → включить Voice Vault Search**.

### 3. Зависимости — двойной клик `setup.bat`
Создаст окружение и поставит библиотеки (несколько минут).
*(Mac/Linux: `bash setup.sh`.)*

### 4. Ollama (для умного поиска)
Установи с [ollama.com/download](https://ollama.com/download) — обычный установщик,
дальше сама работает в фоне.

### 5. Подготовка базы — двойной клик `prepare_base.bat`
**Закрой Obsidian перед этим.** Скрипт скачает модели (~1 + ~4.7 ГБ, один раз) и
допишет в заметки блоки «Возможные вопросы». Идёт несколько минут — это разово.

### 6. Открой Obsidian → **Reindex**
Нажми кнопку Reindex в панели плагина (иконка микрофона на левой панели).
Готово — говори вопрос, откроется заметка.

---

## Первый запуск
При первом старте докачаются модели распознавания (~1.6 ГБ) и поиска — нужен интернет
один раз. Дальше всё работает офлайн.

## Без GPU
Работает и на CPU, просто распознавание медленнее (несколько секунд). С NVIDIA GPU —
быстро. (Опциональное ускорение torch описано в конце README.)

## Если что-то не так
- **Не распознаёт речь / нет реакции** — проверь, что `setup.bat` отработал без ошибок
  и в Obsidian включён плагин. Перезагрузи Obsidian (Ctrl+R).
- **Поиск мажет / ищет по постороннему звуку** — в настройках плагина источник звука
  поставь **Микрофон** (mic), если вопросы говоришь ты; или **Система** (system), если
  вопрос звучит из динамиков (созвон).
- **`prepare_base.bat` пишет, что нет Ollama** — доустанови Ollama (шаг 4) и запусти снова.
- **Речь обрывается на середине** — в настройках увеличь «VAD silence» (напр. до 1300).
